Sample Snowflake setup
======================

Files created:
- sample_db.sql: creates a sample database, schema, table, inserts sample data, and queries it.

How to run:
1. Install the Snowflake CLI if needed.
2. Connect using your Snowflake account.
3. Run:
   snowsql -f sample_db.sql

If you prefer, you can also paste the SQL from sample_db.sql into the Snowflake worksheet.
